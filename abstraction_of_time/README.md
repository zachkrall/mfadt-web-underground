# MFADT: Web Underground
### Abstraction of Time @ Outfront Media

----

Install dependencies
```sh
npm install
```

Start local dev server
```sh
npm run dev
```

Build compiled file for Outfront Media LiveBoard
```sh
npm run build
```

---

Keyboard Shortcuts

| Key | Action |
|---|---|
| R | reset camera |
| E | elevated view |
| D | toggle debug view |
